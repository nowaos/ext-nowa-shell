# Nowa Shell

Modern GNOME Shell interface customizations for GNOME Shell 48.

## Installation

```bash
chmod +x bin/install.sh
./bin/install.sh
```

## Development

### Debug Logs
```bash
./bin/debug.sh
```

### Uninstall
```bash
./bin/uninstall.sh
```

## Code Style

This project follows StandardJS style:
- 2 spaces for indentation
- No semicolons
- Single quotes
